> em-map是一套基于Vue 2.0和高德地图的地图组件。

## npm

推荐 npm 安装。

```
npm install em-map --save
```

## CDN

目前可通过 [unpkg.com/em-map](https://unpkg.com/em-map/dist/index.js) 获取最新版本的资源。

```html
<script src="https://unpkg.com/em-map/dist/index.js"></script>
```
